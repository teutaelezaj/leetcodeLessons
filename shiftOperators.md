# Shift Operators

**left shift**
```
int a = 8; //1 0 0 0
int b = a << 2; // 1 0 0 0 0 0 -> adds 2 zeros, value is now 32
 
System.out.println(b)
Output = 32
```

**right shift**
```
int a = 8; //1 0 0 0
int b = a >> 2; // 1 0 -> removes 2 zeros, value is now 2
 
System.out.println(b)
Output = 2


```
